# Vibe Coder Portfolio

A one-page portfolio website in a **neo-noir / editorial** style focused on atmosphere, typography, and interaction.

## Preview

- Hero section with typing text animation
- Custom GSAP target cursor
- Scroll-driven stacked testimonials
- Fully responsive layout (desktop/tablet/mobile)

## Current version

The active version is a static site built with:

- `index.html`
- `styles.css`
- `script.js`

Hero image:

- `_7qmqmphxgrxlgdz5tnd3_1.png`

## Tech stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- GSAP (CDN)

## Run locally

### Option 1 (fast)

Open `index.html` directly in your browser.

### Option 2 (recommended)

Use any static server (for example, Live Server in VS Code).

## Customization

- Content and section structure: `index.html`
- Visual style and responsiveness: `styles.css`
- Animations (typing, testimonials, cursor): `script.js`

## Deployment

You can deploy this static site to:
