# Vibe Coder Portfolio

Одностраничный портфолио-сайт в стиле **neo-noir / editorial** с акцентом на атмосферу, типографику и интерактив.

## Preview

- Hero с анимацией печати текста
- Кастомный target-cursor на GSAP
- Анимированный стек отзывов при скролле
- Адаптивная верстка для desktop/tablet/mobile

## Текущая рабочая версия

Основная (актуальная) версия сайта сейчас находится в статических файлах:

- `index.html`
- `styles.css`
- `script.js`

## Запуск

### Вариант 1 (быстро)

Просто открой `index.html` в браузере.

### Вариант 2 (через локальный сервер)

Используй любой статический сервер (например, Live Server в VS Code).

## Кастомизация

- Контент и структура секций: `index.html`
- Визуальный стиль и адаптив: `styles.css`
- Анимации (typewriter, отзывы, курсор): `script.js`

## Деплой

Сайт можно разместить как статический проект на:
